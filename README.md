### 🛠️ What I do

I build the boring-but-critical plumbing that lets risky things happen safely on-chain:
**perpetual-futures margin engines, liquidation logic, oracle adapters, and the off-chain risk
models that keep a shared liquidity vault from going insolvent.** Rust or Solidity, whichever the
problem calls for — always with checked math and an adversarial mindset.

Lately that means giving the long tail of on-chain assets something it never had — a real
derivatives layer.

### 🔥 Currently building

> **[Percolator](https://github.com/devharrisonlynch/percolator)** — a permissionless perpetuals
> protocol that makes *every* graduated Pump.fun coin leverage-tradable up to 5×, priced by a
> manipulation-resistant oracle and backed by a single shared HLP vault. Anchor program + typed SDK +
> risk engine + a Next.js trading terminal.
